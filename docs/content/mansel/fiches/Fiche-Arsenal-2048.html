    État du manuscrit

    Marques de restauration modernes : Aucune?
    Qualité du parchemin : Le papier est très endommagé à plusieurs endroits. Dans certains cas, cela empêche la lecture du texte. Parfois le papier a été réparé pour le solidifier :
    - Folios où le papier empêche la lecture : C, 104, 214.
    - Folios où le papier est endommagé ou déchiré mais ne touche pas au texte : A, B, 10, 33, 34, 52, 63, 160, 213.
    Qualité de l’encre :
    - Couleur, nuance et aspect : Noire.
    - Bavures : Très rares.
    - Écaillements ou effacements : Très rares. Mais le papier lui est très endommagé ce qui complique la lecture.
